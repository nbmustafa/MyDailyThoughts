Great choice! Here's how to build a K8s cost recommendation automation using Claude's API:

## The Core Approach

You feed Claude your cluster data (resource requests, limits, usage metrics) and it returns actionable cost-saving recommendations.

---

## 1. Gather K8s Data

Use `kubectl` to pull resource info:

```bash
# Get all pod resource requests/limits
kubectl get pods -A -o json > pods.json

# Get node capacity and usage
kubectl top nodes > node_usage.txt
kubectl top pods -A > pod_usage.txt

# Get HPA, deployments, etc.
kubectl get deployments -A -o json > deployments.json
```

---

## 2. Call Claude API with the Data

```python
import anthropic
import json

client = anthropic.Anthropic()  # uses ANTHROPIC_API_KEY env var

with open("pods.json") as f:
    pods_data = f.read()

with open("node_usage.txt") as f:
    node_usage = f.read()

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=2000,
    system="""You are a Kubernetes cost optimization expert. 
    Analyze cluster data and return specific, actionable recommendations.
    Focus on: over-provisioned resources, idle workloads, right-sizing, 
    spot/preemptible node opportunities, and namespace waste.""",
    messages=[
        {
            "role": "user",
            "content": f"""Analyze this Kubernetes cluster and recommend cost savings:

PODS DATA:
{pods_data[:5000]}  # truncate if large

NODE USAGE:
{node_usage}

Return recommendations as:
1. Quick wins (low effort, high savings)
2. Medium-term optimizations
3. Estimated monthly savings per item
"""
        }
    ]
)

print(response.content[0].text)
```

## 3. Automate It (CronJob or CI)

Run it on a schedule as a Kubernetes CronJob:

```yaml
apiVersion: batch/v1
kind: CronJob
metadata:
  name: claude-cost-advisor
spec:
  schedule: "0 9 * * 1"  # Every Monday at 9am
  jobTemplate:
    spec:
      template:
        spec:
          containers:
          - name: advisor
            image: python:3.11
            env:
            - name: ANTHROPIC_API_KEY
              valueFrom:
                secretKeyRef:
                  name: anthropic-secret
                  key: api-key
            command: ["python", "/scripts/cost_advisor.py"]
          restartPolicy: OnFailure
```

## 4. Common Cost-Saving Signals to Feed Claude

| Data Source | Tool |
|---|---|
| Resource requests vs actual usage | `kubectl top` + Prometheus |
| Unused PVCs | `kubectl get pvc -A` |
| Oversized nodes | `kubectl describe nodes` |
| Unscheduled pods | `kubectl get pods --field-selector=status.phase=Pending` |
| Cost per namespace | Kubecost or OpenCost API |
